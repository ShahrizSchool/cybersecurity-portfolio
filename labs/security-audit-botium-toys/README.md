# Botium Toys Security Audit

## Objective
This lab focuses on performing a basic cybersecurity audit using the NIST Cybersecurity Framework concepts.

The exercise involved reviewing an organization's assets, identifying risks, and evaluating existing security controls.

## Key Areas Analyzed
- Asset management
- Access control
- Encryption
- Password policies
- Backup and disaster recovery

## Findings
Several security gaps were identified including lack of encryption for credit card data, weak password policies, absence of backups, and missing intrusion detection systems.

## Recommendations
Recommended improvements included implementing least privilege, enforcing separation of duties, encrypting sensitive data, strengthening authentication policies, and establishing regular backups.
