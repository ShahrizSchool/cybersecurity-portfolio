# Home Lab – Week 1 (Environment Setup)

## Overview
This lab establishes a safe, isolated home lab environment consisting of
two virtual machines: an attacker system and a defender system configured
on a host-only network. The goal for Week 1 was to verify connectivity
between the two machines without performing any scanning or attacks.

## Lab Status
- Network isolation: VirtualBox host-only
- Connectivity: Verified via ICMP (ping)
- Attacks performed: None
- Environment: Ready for future security labs

➡️ Detailed setup and verification steps can be found in `report.md`.
